# 📊 Tech Layoffs Analysis & Prediction System

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](YOUR_NOTEBOOK_LINK)

This project analyzes **global technology sector layoffs** using **Power BI dashboards and machine learning models**.  
The goal is to explore historical layoff data, visualize trends, and build a predictive model that estimates potential layoffs based on company attributes.

---

# 📌 Project Overview

The tech industry has experienced significant workforce changes in recent years.  
This project combines **data visualization** and **machine learning** to analyze layoff patterns and generate insights.

The project includes:

- 📊 Interactive **Power BI dashboard**
- 🤖 **Machine learning model** for layoff prediction
- 📈 Feature importance analysis
- 📂 Dataset sourced from Kaggle

---

# 📊 Dashboard Features

The Power BI dashboard provides insights such as:

- Total layoffs overview
- Layoffs by industry
- Layoffs by country (map visualization)
- Top 10 companies with the highest layoffs
- Layoff trends over time

These visualizations help identify patterns in workforce reductions across the global tech industry.

---

# 🤖 Machine Learning Model

A **Random Forest Regression model** was developed using **Python and Scikit-learn** to predict the number of layoffs based on company characteristics.

### Model Performance

**R² Score:** 0.53  

The model explains approximately **53% of the variation in layoffs**, which is reasonable for real-world economic data where multiple external factors influence layoffs.

### Key Features Used

- Industry
- Country
- Funding Raised
- Company Stage
- Percentage of Workforce Laid Off
