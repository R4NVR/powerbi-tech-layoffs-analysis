# Power BI Tech Layoffs Analysis

This project analyzes global tech layoffs using Power BI dashboards.

## Features
- Total layoffs visualization
- Layoffs by industry
- Layoffs by country map
- Top 10 companies with highest layoffs
- Layoff trends over time

## Tools Used
- Power BI
- Data Visualization
- CSV dataset

## Dataset
Tech Layoffs Dataset (Kaggle)
